# 全国パン図鑑

## サイト概要  
全国のパン屋さんのパンレビューサイト  

### サイトテーマ  
パン屋さんのおいしいパンを共有する  

### テーマを選んだ理由  
パンのおいしさをもっと広めたい

### ターゲットユーザ  
パンが好きな人、興味がある人  

### 主な利用シーン  
パン屋さんに行きたいと思ったとき,おいしいパンを共有したいとき
